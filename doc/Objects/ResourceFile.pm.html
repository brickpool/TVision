NAME

TResourceFile - Implements a stream that can be indexed by key strings.

SYNOPSIS

  use TurboVision::Objects;
  ...
    my ($res_file, $my_menu);
    
    # construct the menu bar as usual
    $my_menu =  .... 
    $res_file = TResourceFile->Init(
      TBufStream->init( 'menu.sav', ST_CREATE, 1024 )
    );

    # Put the menu bar and name the resource "mainmenu"
    $res_file->put($my_menu, 'mainmenu');

    # Flush and close
    undef $res_file;
  }

DESCRIPTION

TResourceFile is a special purpose random access TStream that let's you find records by a key string instead of a record number.

Using TResourceFile, you can implement simple data base retrievals, as well as store Turbo Vision components.

Commonly Used Features

In addition to "init", "put" and "get" are the most frequently used methods of the TResourceFile object.

Attributes

stream
  param stream ( is => rwp, type => TStream );

Points to the stream used by this TResourceFile object.

modified
  has modified ( is => rwp, type => Bool );

If the file has been modified, then this flag is set to true.

The "flush" method checks this flag to determine if it should update the resource file.

Constructors

init
  factory init(TStream $a_stream) : TResourceFile

TResourceFile->init is called after opening a stream.

The opened stream is passed as a parameter to init and becomes the stream that holds the resource file.

Destructors

DEMOLISH
  method DEMOLISH()

Calls "flush" and destroys the (private) index collection and the resource "stream" file.

Methods

count
  method count() : Int

Computes and returns the number of items or resources stored in the file.

delete
  method delete(Str $key)

Removes the $key from the index and marks the space it previously occuppied as being deleted.

See "switch_to" for a method to reclaim the now unused space.

flush
  method flush()

Checks the "modified" flag, and if true, updates the resource stream file, and then resets "modified" to false.

get
  method get(SimpleStr $key) : Object|Undef

Uses $key as an index into the resource and returns a pointer to the object that it references, or undef if the $key is not in the file.

See: "put"

key_at
  method key_at(Int $i) : Str

Use key_at to scan through the entire resource file.

The parameter $i is an index to each resource in the file, numbered 0 to "count" minus 1.

The method key_at returns the string corresponding to the key value at the $i'th index position.

put
  method put(Object $item, Str $key)

Stores the object pointed to by $item into the resource file, using the specified $key.

See: "get"

switch_to
  method switch_to(TStream $a_stream, Bool $pack) : TStream|Undef

Use switch_to to copy the current resource file to another stream specified by $a_stream.

If $pack is true, switch_to will not copy objects marked as deleted, thereby compressing the resulting resource file.

See: "delete"
