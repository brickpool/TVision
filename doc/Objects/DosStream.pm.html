NAME

TDosStream - Implements a file stream class

SYNOPSIS

The use of TDosStream is nearly identical to TBufStream, without the buffering related methods. See the TBufStream example.

DESCRIPTION

Use TDosStream for unbuffered stream file access (see also TBufStream). For most applications you will probably prefer to use TBufStream for its faster, buffered file access.

Generally, you will use the "init" constructor to open a stream file for access, the "read" and "write" methods for performing input and output. "DEMOLISH" will to close the open stream of the object. For random access streams, you will use "seek" to position the file pointer to the proper object record.

Attributes

handle
  param handle ( is => rwp, type => FileHandle, predicate => '_is_openhandle' );

handle contains the OS file handle used to access the file containing the stream.

Constructors

init
  factory init(Str $file_name, Int $mode) : TDosStream

This constructor creates a OS file stream with the given $file_name. The $mode argument must be one of the values ST_CREATE, ST_OPEN_READ, ST_OPEN_WRITE, or ST_OPEN. These constant values are part of the TStream class.

Destructors

DEMOLISH
  sub DEMOLISH()

Closes the file of the stream object.

Methods

read
  around read(Item $buf, Int $count)

Use read to read $count bytes from the stream and into the $buf parameter. read begins at the stream's current position (as determined by seek or beginning at the end of the previous read operation).

seek
  around seek(Int $pos)

Positions the current stream position to $pos. You can use seek to implement random access to stream files.

truncate
  around truncate()

Use truncate to delete all data following the current position in the stream.

write
  around write(Str $buf, Int $count)

Use write to copy $count bytes from the $buf parameter to the stream.
