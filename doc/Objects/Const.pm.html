NAME

TurboVision::Objects::Const - Constants used by Objects

SYNOPSIS

  use TurboVision::Objects::Const qw(
    MAX_COLLECTION_SIZE
    :stXXXX
  );
  ...

EXPORTS

Nothing per default, but can export the following per request:

  :all
    MAX_COLLECTION_SIZE
  
    :coXXXX
      CO_INDEX_ERROR
      CO_OVERFLOW
  
    :stXXXX
      ST_OK
      ST_ERROR
      ST_INIT_ERROR
      ST_READ_ERROR
      ST_WRITE_ERROR
      ST_GET_ERROR
      ST_PUT_ERROR
      ST_CREATE
      ST_OPEN_READ
      ST_OPEN_WRITE
      ST_OPEN
