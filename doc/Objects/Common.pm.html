NAME

TurboVision::Objects::Common - Routines and tools used by Objects

SYNOPSIS

  use 5.014;
  use TurboVision::Objects::Common qw( :all );
  
  sub init {
    my ($class, @args) = @_;
    return fail() if @args != 1;
    ...
  }
  ...
  sub get_item {
    abstract();
    return;
  }
  ...
  say long_div(5, 2);
  say long_mul(5, 2);
  ...
  say word_rec($w)->lo;
  say long_rec($l)->hi;
  say ptr_rec($p)->ofs;
  ...

EXPORTS

Nothing per default, but can export the following per request:

  :all
  
    :tools
      byte
      integer
      longint
      long_div
      long_mul
      long_rec
      ptr_rec
      word
      word_rec
    
    :subs
      abstract
      dispose_str
      fail
      new_str
      register_objects

DESCRIPTION

Routines and tools that are used in all modules and, if applicable, in the program and have been assigned to the Objects module.

Tools

byte
  func byte() : Object
  func byte(Str|Int $value) : Object

The utility byte helps to convert an unsigned char (octet) value into a number to the base 256 (packed string).

A byte data type support only positive Int values in a range of: 0..255.

Usage:

  $ = byte()->type;       # uint8 template for the conversion
  $ = byte()->size;       # Size of a uint8 value in bytes

  $ = byte($int)->cast;   # Conversion of an integer to a uint8 value
  $ = byte($int)->pack;   # Conversion from uint8 to a number in base 256
  $ = byte($str)->unpack; # Conversion of a number with base 256 into uint16

long_div
  func long_div(Int $x, Int $y) : Int

A division routine, returning the integer value int( $x/$y ).

Note: This utility routine is for compatiblity only.

long_mul
  func long_mul(Int $x, Int $y) : Int

A multiplication routine, returning the integer value $x*$y.

Note: This utility routine is for compatiblity only.

long_rec
  func long_rec(Int $l) : Object|Value

A utility routine allowing access to the long_rec($l)->lo and long_rec($l)->hi words of $l. In a scalar context the utility routine returns $l & 0xffffffff.

Note: This utility routine is for compatiblity only. Please use $hi = $int & 0xfffff0000 to get the hi-word or $lo = $int & 0x0000fffff to get lo-word of an integer value.

ptr_rec
  func ptr_rec(Ref $p) : Object|Value

A utility routine that returns the memory address of a referenced value as ptr_rec($p)->seg and ptr_rec($p)->ofs. In a scalar context the utility routine returns refaddr($p).

Note: Modern systems are not segmented and the addresses are always linear. This utility routine is only for compatiblity. Please use Perl's routine refaddr instead to get the internal memory address of a referenced value.

See: refaddr from Scalar::Util

word_rec
  func word_rec(Int $w) : Object|Value

A utility routine allowing access to the word_rec($w)->lo and word_rec($w)->hi bytes of $w. In a scalar context the utility routine returns $w & 0xffff.

Note: This utility routine is for compatiblity only. Please use $hi = $int & 0xff00 to get the hi-byte or $lo = $int & 0x00ff to get lo-byte of an (unsigned) integer value.

Routines

abstract
  func abstract()

Terminates program with a run-time error 211. When implementing an abstract object type, call abstract in those methods that must be overridden in descendant types. This ensures that any attempt to use instances of the abstract object type will fail.

See: confess from Carp

dispose_str
  func dispose_str(ScalarRef[Str] $s)

Weaken the string reference $s.

Note: This utility routine is for compatiblity only.

See: weaken from Scalar::Util

fail
  func fail() : Undef

fail (which calls FAIL from module Contextual::Return) can be used in a constructor for an object or class. It will exit the constructor immediately.

This means that calling fail inside new (also applies to init and load in this library) raise an exception or returns false (undef) when used in a boolean context.

See: FAIL from Contextual::Return

new_str
  func new_str(Str $s) : Ref|Undef

If $s is empty, new_str returns undef. Otherwise, new_str returns a reference to a copy of string $s.

Note: This utility routine is for compatiblity only.

register_objects
  func register_objects()

Calls register_type for each of the class types defined in the Objects module: TCollection, TStringCollection
