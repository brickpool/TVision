NAME

TResourceCollection - Implementing a collections of resources.

SYNOPSIS

See the TStringCollection example.

DESCRIPTION

Several specialized collections are derived from TCollection, including TResourceCollection.

The TResourceCollection is used internally to the resource file mechanism and is not generally used by application programs.

Methods

key_of
  around key_of(TResourceItem $item) : ScalarRef[SimpleStr]

key_of returns a string reference to the key of the $item.

get_item
  around get_item(TStream $s) : TResourceItem

Uses the TStream->read function to read a TResourceItem hash from stream $s and returns a hash reference of type TResourceItem.

put_item
  around put_item(TStream $s, TResourceItem $item)

By default, writes a hash referenced by $item to the stream $s by calling TStream->write.
