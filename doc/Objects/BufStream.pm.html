NAME

TBufStream - Implements a buffered version of TDosStream.

SYNOPSIS

  my $phone_book;
  my $phone_book_file;
  
  package TPersonInfo {
    use Moose;
    use Scalar::Util qw( blessed );
    use TurboVision::Objects;

    extends TCollection->class;
    
    ...
    
    # TPersonInfo is a collection object holding name and
    # address information
    sub store {
      my ($self, $s) = @_;
      confess 'Invalid argument'
        if !blessed($self)
        || !blessed($s);
      s->write_str( $self->name );
      s->write_str( $self->address );
      s->write_str( $self->city );
      s->write_str( $self->state );
      s->write_str( $self->zip );
      s->write_str( $self->age );
    }
    
    # constructor
    sub load {
      my ($class, $s) = @_;
      confess 'Invalid argument'
        if !defined($class)
        || !blessed($s);
      return fail
          ref($class);
      my $name    = s->read_str();
      my $address = s->read_str();
      my $city    = s->read_str();
      my $state   = s->read_str();
      my $zip     = s->read_str();
      my $age     = s->read_str();
      return $class->new(
        name    => $name,
        address => $address,
        city    => $city,
        state   => $state,
        zip     => $zip,
        age     => $age,
      );
    }
  }
  
  ...
  
  # Register the PersonInfo object type
  register_type( RPersonInfo );
  
  # Open the stream file
  $phone_book_file = TBufStream->Init( 'fonebook.dat', ST_CREATE, 1024 );
  
  # Tell the $phone_book collection to put itself to the stream
  $phone_book_file->put( $phone_book );
 
  # The file is closed by the DEMOLISH method
  undef $phone_book_file;

DESCRIPTION

The TBufStream object type is a derivative of TStream and TDosStream and is probably the type you will most often use for writing and reading streams to and from disk files. TBufStream implements a buffered version of TDosStream, which greatly improves the speed and efficiency of stream I/O, particularly when writing or reading a lot of small objects.

Generally, you will use the TBufStream-init method to open a stream file for access, the TBufStream->read and TBufStream->write methods for performing input and output. For random access streams, you will use TBufStream->seek to position the file pointer to the proper object reocrd. You may also wish to use TBufStream->flush or TBufStream->truncate, as appropriate.

public readonly Int buf_end

If the buffer is not full, buf_end gives an offset to the last used byte in the buffer.

public readonly Str buffer

Stream buffer as byte packed string.

public readonly Int buf_ptr

An offset from the buffer string indicating the current position.

The size of the buffer in bytes.

Constructors

public TBufStream->init(Str $file_name, Int $mode)

Constructs the object and opens the file named in $file_name with access mode $mode by calling the init constructor inherited from TDosStream.

Destructors

public DEMOLISH()

Calls flush to flush buffer contents to disk.

Methods

public flush()

Flushes the stream's buffer provided the stream's status is stOK.

public read(Item $buf, Int $count)

If the stream's status is stOK, reads $count bytes into the $buf.

public seek(Int $pos)

Sets the current position to $pos bytes from the beginning of the stream.

public truncate()

truncate deletes all data on the calling stream from the current position.

public write(Str $buf, Int $count)

Writes $count bytes from the $buf buffer to the stream, starting at the current position.
