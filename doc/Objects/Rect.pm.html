NAME

TRect - Defines two coordinates of a rectangle.

SYNOPSIS

  use TurboVision::Objects;
  
  my $bounds = TRect->new();
  my $a_rectangle;
  ...
  # Initialize to the coordinates (0,0) and (80,2)
  $bounds->assign(0, 0, 80, 2);
  ...
  # Copy the contents of $bounds to $a_rectangle
  $a_rectangle->copy( $bounds );

DESCRIPTION

TRect is a simple object representing a rectangle on the screen.

TRect primarly defines two coordinates, "a" and "b", which are the upper left and the lower right corners of a rectangle. TRect parameters are used throughout Turbo Vision to specify the screen location and size of windows, dialog boxes and entry fields.

Class

public class TRect

TRect is a standalone object.

Attributes

a
  has a ( is => rw, type => TPoint ) = TPoint->new;

a is the point defining the top left corner of a rectangle on the screen.

b
  has b ( is => rw, type => TPoint ) = TPoint->new;

b is the point defining the bottom right corner of a rectangle on the screen.

equals

Returns true if $r is the same as the rectangle.

grow
  method grow(Int $a_dx, Int $a_dy)

Changes the size of the rectangle by subtracting $a_dx from a->x, adding $a_dx to b->x, subtracting $a_dy from a->y, and adding $a_dy to b->y.

intersect
  method intersect(TRect $r)

Changes the location and size of the rectangle to the region defined by the intersection of the current location and that of $r.

move
  method move(Int $a_dx, Int $a_dy)

Moves the rectangle by adding $a_dx to a->x and b->x and adding $a_dy to a->y and b->y.

union
  method union(TRect $r)

Changes the rectangle to be the union of itself and the rectangle $r.
