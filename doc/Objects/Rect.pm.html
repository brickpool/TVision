NAME

TRect - Defines two coordinates of a rectangle.

SYNOPSIS

  use TurboVision::Objects;
  
  my $bounds = TRect->new();
  my $a_rectangle;
  ...
  # Initialize to the coordinates (0,0) and (80,2)
  $bounds->assign(0, 0, 80, 2);
  ...
  # Copy the contents of $bounds to $a_rectangle
  $a_rectangle->copy( $bounds );

DESCRIPTION

TRect is a simple object representing a rectangle on the screen.

TRect primarly defines two coordinates, a and b, which are the upper left and the lower right corners of a rectangle. TRect parameters are used throughout Turbo Vision to specify the screen location and size of windows, dialog boxes and entry fields.

Class

TRect is a standalone object.

public TPoint a

Methods

Object Methods

public assign(Int $ax, Int $ay, Int $bx, Int $by)

Assigns the parameter values to the rectangle's point attributes. $ax becomes a->x, $bx becomes b->x, and so on.

public Bool contains(TPoint $p)

Returns true if the rectangle contains the point $p.

public copy(TRect $r)

copy sets all attributes equal to those in rectangle $r.

public Bool empty()

Returns true if the rectangle is empty

public Bool equals(TRect $r)

Returns true if $r is the same as the rectangle.

public grow(Int $aDX, Int $aDY)

Changes the size of the rectangle by subtracting $aDX from a->x, adding $aDX to b->x, subtracting $aDY from a->y, and adding $aDY to b->y.

public intersect(TRect $r)

Changes the location and size of the rectangle to the region defined by the intersection of the current location and that of $r.

public move(Int $aDX, Int $aDY)

Moves the rectangle by adding $aDX to a->x and b->x and adding $aDY to a->y and b->y.

public union(TRect $r)

Changes the rectangle to be the union of itself and the rectangle $r.
