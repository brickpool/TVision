NAME

Types - Types for Objects

THE TYPES

Basic Types

public type FNameStr

OS file name string.

public type PString

Defines a reference to a string.

public type TResourceItem

This internal Dict type is used by TResourceCollection.

public type TItemList

This is an internal object type used for maintaining an array of references in TCollection objects.

public type TStrIndexRec

This internal Dict record type is used by TStrIndex.

public type TStrIndex

This internal ArrayRef type of TStrIndexRec is used by TStringList and TStrListMaker.
