NAME

TStream - General abstract class providing I/O to and from a storage device.

SYNOPSIS

See the TBufStream example.

DESCRIPTION

TStream is the root of the various stream objects. For your applications, you must use the descendants, TDosStream, TBufStream or TMemoryStream.

Class Attributes

stream_error
  class_has stream_error (
    is        => rw,
    type      => CodeRef,
    predicate => 'has_stream_error',
    clearer   => 'clear_stream_error',
  );

stream_error allows you to globally override all stream error handling. For this purpose, an subroutine reference is assigned to the class variable. A TStream object is passed as a parameter.

  package main;

  TStream->stream_error( sub {
    printf "Catch stream error info #%d\n", shift->error_info
  } );
  ...
  $stream->error(ST_ERROR, 1);  # 'Catch stream error info #1'

See module "MooseX::ClassAttribute" for class_has.

Object Attributes

error_info
  has error_info ( is => rw, type => Int ) = 0;

Whenever an error has occurred (indicated by "status" != ST_OK), the error_info attribute contains additional information.

For some values these will be additional stXXXX constants, although for OS-level file errors, error_info will contain the system errno code (like $! use).

status
  has status ( is => rw, type => Int ) = ST_OK;

Holds either ST_OK or the last error code. See the stXXXX constants for possible status values.

Methods

copy_from
  method copy_from(TStream $s, Int $count)

This method copies $count bytes from stream $s to the current stream object.

error
  method error(Int $code, Int $info)

The various stream methods call error when a problem is encountered. The method error, in turn, calls the subroutine referenced to the class variable "stream_error" (if its not undef).

Once an error condition has been raised, you must call "reset" before performing additional stream reads or writes.

flush
  method flush()

TStream->flush has no operation, but is overridden in by descendant objects, such as TBufStream, in order to write the buffer to disk.

get
  method get() : Object|Undef

Reads an object from the stream, interpreting the first bytes as the TStreamRec->obj_type attribute of a registeration record.

TStreamRec->obj_type is used to determined exactly what type of object this is, and then to call that object's load constructor to create a new instance.

The method get returns a refernce to the newly created Object, or undef on error.

get_pos
  method get_pos() : Int;

The method get_pos compute the current byte location within the stream.

It returns the stream's current position, or -1 on error.

get_size
  method get_size() : Int

The method get_size computer the total number of bytes in the stream.

It returns the total size, or -1 on error.

put
  method put(Object $p)

Writes the TStreamRec->obj_type attribute of the registration record to the stream, and calls the object's store method to write the appropriate data attributes.

If the object type of $p has not been registered, put calls "error" and doesn't write anything to the stream.

read
  method read(Item $buf, Int $count)

All descendants of TStream override this procedure to copy $count bytes from the stream into $buf, and to move the current stream position to the next object position.

read_str
  method read_str() : SimpleStr|Undef

Reads a string from the stream and returns a simple string (a string with less than 256 characters without any control character) from the current position of the stream, or undef on error.

This function is used by TStringCollection->get_item to read a string from the stream.

reset
  method reset()

Clears any existing error conditions.

seek
  method seek(Int $pos)

The method seek sets the stream's current position to the byte offset specified by $pos.

str_read
  method str_read() Str|Undef

Reads a string from the current position of the calling stream, returning a Str.

returns Str, or undef on error.

str_write
  method str_write(Str $p)

Writes a string $p to the calling stream, starting at the current position.

truncate
  method truncate() >>

All descendants must override truncate to delete all data in the stream after the current position.

write
  method write(Str $buf, Int $count)

All descendants must override this method to copy $count bytes from $buf and write them to the stream.

write_str
  method write_str(SimpleStr $p)

Outputs a simple string (a string with less than 256 characters without any control character) $p to the current stream.
