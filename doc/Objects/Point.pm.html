NAME

TPoint - Defines a point on the screen.

SYNOPSIS

  use TurboVision::Objects;
  
  # Defines the minimum size of a window object
  my $min_win_size = TPoint->new(x => 16, y => 6);

DESCRIPTION

TPoint is a simple object representing a point on the screen.

Class

public class TPoint

TPoint is a standalone object.

Attributes

public Int x

x is the screen column of the point.

public Int y

y is the screen row of the point.

Methods

public copy(TPoint $pt)

copy sets all attributes equal to those in point $pt.

Overload Methods
