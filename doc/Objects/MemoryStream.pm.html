NAME

TMemoryStream - Implements a stream that stores data in memory

DESCRIPTION

TMemoryStream is a stream that stores its data in dynamic memory.

Constructors

public TMemoryStream->init(Int $a_limit, Int $a_block_size)

This Constructor creates an memory stream with the given minimum size in bytes. Calls the init constructor inherited from TStream, then sets block_size and alocate the memory.

Methods

public read(Item $buf, Int $count)

Reads $count bytes from the stream, starting at the current position, into the $buf buffer.

public truncate()

truncate deletes all data on the calling stream from the current position.

public write(Str $buf, Int $count)

Writes $count bytes from the $buf buffer to the stream, starting at the current position.
