NAME

TurboVision::Views::Types - Types for Views

SYNOPSIS

  use TurboVision::Views::Types;
  ...

THE TYPES

Basic Types

TCommandSet
  class_type TCommandSet : Object;

In Turbo Vision, command codes are assigned values from 0 to 65535, with values in the range of 0 to 255 reserved for items that can be selectively disabled.

TCommandSet is used to hold a set of up to 256 commands, specifically those that can be disabled, and is used as a parameter for the TView methods, enable_commands, disable_commands, get_commands and set_commands.

TDrawBuffer
  subtype TDrawBuffer : ArrayRef[Ref];

TDrawBuffer is used to create temporary storage areas for a line of text to be written to the screen where the low byte of each word contains the character value, and the high byte contains the video attribute byte.

You can use the Turbo Vision sub routines, move_buf, move_char, move_c_str, and move_str to set up the buffer and then use one of the TView method's write_buf and write_line within a draw method to write the output to the screen.

Here's an example using TDrawBuffer:

  my $buffer = [];
  ...
  move_str($buffer, 'Financial Results for FY1991', get_color(1) );
  $self->write_line( 1, 3, 28, 1, $buffer );

See: TView methods write_buf and write_line.

TFixupList
  subtype TFixupList : ArrayRef[Ref];

Fix up reference array.

TPalette
  subtype TPalette : Str;

Defines the data type used for storing color palettes.

Since all color palettes are equivalent to strings, you can use, if you wish, all of the various string manipulation functions, including indexing (with substr), copy (with =), delete (undef), insert (also substr) and so on.

TTitleStr
  subtype TTitleStr : Str;

Defines a type used by TWindow for window title strings.

TVideoBuf
  subtype TVideoBuf : ArrayRef[Int];

This defines the internal type used in video buffer declarations in TGroup.

Video buffers are used to store screen images in cache memory (see get_buf_mem) for rapid screen update.

TScrollChars
  subtype TScrollChars : Str;

This is an internal type used inside TScrollBar to store the characters used to draw a TScrollBar object on the display.
