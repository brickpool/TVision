NAME

TurboVision::Views::Common - Variables and tools used by Views

SYNOPSIS

  use TurboVision::Views::Common qw(
    :vars
  );
  ...

EXPORTS

Nothing per default, but can export the following per request:

  :all

    :vars
      $command_set_changed
      $cur_command_set
      $shadow_size

    private:
      $_fixup_list
      $_owner_group
      $_the_top_view

Variables

$command_set_changed
  our $command_set_changed = < Bool >;

True if the command set has changed since being set to false.

$cur_command_set
  our $cur_command_set = < Str >;

$cur_command_set is a string created with Perl's command vec.

It stores the current command set. By default, all commands except those of the window are active.

$shadow_size
  our $shadow_size = < TPoint >;

Shadow sizes.

$_fixup_list
  our $_fixup_list = < TFixupList >;

Used for loading.

$_owner_group
  our $_owner_group = < TGroup >;

Used for loading.

$_the_top_view
  our $_the_top_view = < TView >;

Top focused view.
