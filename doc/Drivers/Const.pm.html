DESCRIPTION

This module is used to define Turbo Vision related constants that are used for the Drivers modules.

For the use of the EventManager module constants for events (evXXXX), keyboard (kbXXXX) and mouse (mbXXXX) are defined here. Furthermore, additional constants for the video display of the ScreenManager are defined here (smXXXX).

Turbo Vision (and also Perl) uses standardized naming conventions as an aid to making source code readable.

The most important conventions for constants are:

  1.  With the exception of the constants for the color palette and the
      registration data sets, all constants are specified completely in capital
      letters.
  2.  All (other) constants defined for the Turbo Vision implementation of Perl
      start with two capital letters and an underscore.
  3.  For better readability, the constants defined here for Turbo Vision use
      underscores to separate parts of words.

The following table shows the constants used here for Turbo Vision with prefix:

  Prefix  Definition               Example
  EV_     Event constant           EV_COMMAND
  KB_     Keyboard code constant   KB_ALT_A
  MB_     Mouse button constant    MB_RIGHT_BUTTON
  SM_     Screen mode constant     SM_FONT8X8

Keyboard Shift State Constants (:kbXXXX)

The kbXXXX constants for detecting keyboard shift states.

The keyboard shift state constants are used as a bit mask to test the status of various keyboard keys, such as the Ctrl or Shift key.

For example, to see if the keyboard is producing shifted characters, declare an subroutine and test the bits like this:

    sub shift_state;
    ...
    if ( shift_state & ( kbRightShift | kbLeftShift | kbCapsState ) ) {
      ...
KB_RIGHT_SHIFT
  constant KB_RIGHT_SHIFT = < Int >;

Bit set if the right shift key down.

KB_LEFT_SHIFT
  constant KB_LEFT_SHIFT = < Int >;

Bit set if the left shift key down.

KB_CTRL_SHIFT
  constant KB_CTRL_SHIFT = < Int >;

Bit set if the Ctrl key is down.

Bit set if the Alt key is down.

Keyboard Scancode Constants (:kbXXXX)

The kbXXXX constants for the non-standard keystrokes such as function and Alt keys.

Use these constants to check for specific keystroke values in the TEvent->key_code field. For example,

  if ( $event->key_code == KB_PG_DN ) {
    # handle page down function
    ...

_SCREEN_RESOLUTION
  constant _SCREEN_RESOLUTION = < CodeRef >;

Hash constants for converting standard video text modes (0x0000..0x00ff, 0x0100..0x07ff or 0x0900..0x09ff) to resolution modes (0x1000..0x7fff).

The code has a 0xHHWW form where HH is a number of rows and WW is a number of columns.

_STANDARD_CRT_MODE
  constant _STANDARD_CRT_MODE = < CodeRef >;

Hash constants for converting resolution modes (0x1000..0x7fff) to standard text modes (0x0000..0x00ff, 0x0100..0x07ff or 0x0900..0x09ff).
