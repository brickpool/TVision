NAME

TurboVision::Drivers::Win32::LowLevel - Windows low level API routines

EXPORTS

Nothing per default, but can export the following per request:

  GWL_STYLE
  WS_SIZEBOX

  GetConsoleWindow
  GetDoubleClickTime
  GetWindowLong
  SetWindowLong

Constants

GWL_STYLE
  constant GWL_STYLE = < Int >;

Parameter for SetWindowLong. Sets a new window style.

WS_SIZEBOX
  constant WS_SIZEBOX = < Int >;

Defines if the window should have a sizing border.

Subroutines

GetConsoleWindow
  func GetConsoleWindow() : Int

Retrieves the window handle from the calling process that is used by the console; for more information consult the original API documentation.

GetDoubleClickTime
  func GetDoubleClickTime() : Int

Retrieves the current double-click time for the mouse; for more information consult the original API documentation.

GetWindowLong
  func GetWindowLong(Int $hWnd, Int $nIndex) : Int

Retrieves a windows property; for more information consult the original API documentation.

SetWindowLong
  func SetWindowLong(Int $hWnd, Int $nIndex, Int dwNewLong) : Int

Sets a windows property; for more information consult the original API documentation.
