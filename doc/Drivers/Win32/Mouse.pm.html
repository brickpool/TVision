NAME

TurboVision::Drivers::Win32::Mouse - Event Manager Mouse implementation

DESCRIPTION

This module implements EventManager routines for the Windows platform. A direct use of this module is not intended. All important information is described in the associated POD of the calling module.

EXPORTS

Nothing per default, but can export the following per request:

  :all

    :mouse
      get_mouse_event
      hide_mouse
      show_mouse

    :private
      _detect_mouse

_detect_mouse
  func _detect_mouse() : Int

Detect mouse driver and set $button_count.

Returns the number of the buttons on your mouse, or zero on errors.
