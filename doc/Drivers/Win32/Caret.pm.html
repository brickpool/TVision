NAME

TurboVision::Drivers::Win32::Caret - Caret functions.

DESCRIPTION

This module implements Caret routines for the Windows platform. A direct use of this module is not intended. All important information is described in the associated POD of the calling module.

EXPORTS

Nothing per default, but can export the following per request:

  :all

    :caret
      done_caret
      get_caret_size
      init_caret
      is_caret_visible
      set_caret_size
      set_caret_position

Subroutines

done_caret
  func done_caret()

This internal routine implements done_caret for Windows.

get_caret_size
  func get_caret_size() : Int

Return the shape of the cursor, like interrupt int 10h function 03h does.

Note: If the cursor is not visible, the routine returns 0x2000 (bit 5 of "Scan Row Start" is set).

init_caret
  func init_caret()

This internal routine implements init_caret for Windows.

is_caret_visible
  func is_caret_visible() : Bool

If the cursor is hidden (bit 5 of "Scan Row Start" in "get_caret_size" is set), the routine returns false, otherwise true ("cursor shown").

See <L/get_caret_size>

set_caret_position
  func set_caret_position(Int $x, Int $y)

Set the cursor position, as interrupt int 10h function 02h do.

set_caret_size
  func set_caret_size(Int $cursor)

Set the shape of the cursor, as interrupt int 10h function 01h do.
