NAME

TurboVision::Drivers::Win32::Keyboard - Event Manager Keyboard implementation

DESCRIPTION

This module implements EventManager routines for the Windows platform. A direct use of this module is not intended. All important information is described in the associated POD of the calling module.

EXPORTS

Nothing per default, but can export the following per request:

  :all
  
    :kbd
      get_key_event

Subroutines

get_key_event
  func get_key_event(TEvent $event)

This internal routine implements get_key_event for Windows; more information about the routine is described in the EventManager module.
