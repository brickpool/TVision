NAME

TurboVision::Drivers::Win32::Screen - Windows Video Display Manager

DESCRIPTION

This module implements ScreenManager routines for the Windows platform. A direct use of this module is not intended. All important information is described in the associated POD of the calling module.

_fix_crt_mode
  func _fix_crt_mode(Int $mode) : Int

Fix CRT mode in $mode if required.

If possible, PC-standard text modes are used. If not, the return of $mode is in the form 0xHHWW, where HH is a number of rows and WW is a number of columns. E.g. 1950h (0x1950) corresponds to a 80x25 mode.
