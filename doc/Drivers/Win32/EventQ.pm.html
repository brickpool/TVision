NAME

TurboVision::Drivers::Win32::EventQ - Event Manager implementation

DESCRIPTION

This module implements EventManager routines for the Windows platform. A direct use of this module is not intended. All important information is described in the associated POD of the calling module.

EXPORTS

Nothing per default, but can export the following per request:

  :all
  
    :events
      init_events
      done_events

    :private
      $_auto_delay
      $_auto_ticks
      $_down_buttons
      $_down_ticks
      $_down_where
      @_event_queue
      $_last_buttons

Variables

EventQ Variables

$_auto_delay
  our $_auto_delay : Int;

Event manager variable for EV_MOUSE_AUTO delay time counter.

$_auto_ticks
  our $_auto_ticks : Int;

Event manager variable for held mouse button tick counter.

$_down_buttons
  our $_down_buttons : Int;

Event manager variable for the current state of the mouse buttons.

$_down_ticks
  our $_down_ticks : Int;

Event manager variable for down mouse button tick counter.

$_down_where
  our $_down_where : TPoint;

Event manager variable for the current state of the mouse position when the mouse button is pressed.

@_event_queue
  our @_event_queue : Array;

Event manager queue for the TEvent records.

$_last_buttons
  our $_last_buttons : Int;

Event manager variable for the previous state of the mouse buttons.

$_last_double
  our $_last_double : Bool;

Event manager variable for the previous state of double klick.

$_last_where
  our $_last_where : TPoint;

Event manager variable for the previous mouse position.

$_shift_state
  our $_shift_state : Int;

Key shift state.

$_ticks
  our $_ticks : Int;

This (magic) variable returns the number of timer ticks (1 second = 18.2 ticks), similar to the direct memory access to the BIOS low memory address 0x40:0x6C.
