NAME

TurboVision::Drivers::Win32::StdioCtl - Implementation of a STD ioctl.

DESCRIPTION

StdioCtl (an abbreviation of Standard input/output control) is a system call for device-specific input/output operations and other operations which cannot be expressed by regular system calls.

StdioCtl is singleton a class that has only one instance in an application. The module is similar to the functionalities of MooseX::Singleton.
