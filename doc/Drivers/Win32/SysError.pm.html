NAME

TurboVision::Drivers::Win32::SysError - System Error Handler implementation

DESCRIPTION

This module implements SystemError routines for the Windows platform. A direct use of this module is not intended. All important information is described in the associated POD of the calling module.

EXPORTS

Nothing per default, but can export the following per request:

  :all

    :syserr
      init_sys_error
      done_sys_error
      system_error
