THE TYPES

Basic Types

MouseEventType
  subtype MouseEventType : Dict[
    buttons => Int, where => TPoint, event_flags => Int
  ];

The MouseEventType is used internally for the hardware mouse driver.

TSysErrorFunc
  subtype TSysErrorFunc : CodeRef;

TSysErrorFunc defines what the system error handler function looks like. See $sys_error_func and system_error for details of intercepting system level errors.

TVideoCell
  subtype TVideoCell : Int;

TVideoCell describes one character on the screen. One of the bytes contains the color attribute with which the character is drawn on the screen, and the other byte contains the ASCII code of the character to be drawn.

The exact position the high byte represents the color attribute, while the low-byte represents the ASCII code of the character to be drawn.

TVideoBuf
  subtype TVideoBuf : ArrayRef[TVideoCell];

The TVideoBuf type represents the screen.

TVideoMode
  subtype TVideoMode : Dict[ col => Int, row => Int, color => Bool ];

The TVideoMode describes a video mode. Its fields are self-explaining: col, row describe the number of columns and rows on the screen for this mode. color is True if this mode supports colors, or False if not.
